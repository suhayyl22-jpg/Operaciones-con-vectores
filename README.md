# Operaciones-con-vectores

## Descripción

Este proyecto contiene ejercicios desarrollados en Python y LaTeX relacionados con:

- Vectores paralelos
- Vectores perpendiculares
- Módulo de un vector
- Ángulo entre dos vectores
- Aplicación práctica usando Python

## Bibliotecas utilizadas

- NumPy
- Math
- Matplotlib

## Archivos del proyecto

- `trabajo_vectores.py`
- `informe_vectores.tex`
- `grafica_vectores.png`
